## Access Controls

Omnipair's programs are deployed using the BPF Upgradeable Loader, enabling the Upgrade Authority to push updates when necessary.

**All program Upgrade and Admin Authority is held under a [Squads multi-sig](https://app.squads.so/squads/4JpPs9Mi11qoj6GthQPiTjUc4gXq8BAoSs3AD6NVjQUZ/home).**

Programs are fully upgradeable (labeled as such on [Solscan](https://solscan.io/account/omnixgS8fnqHfCcTGKWj6JtKjzpJZ1Y5y9pyFkQDkYE)). All changes undergo review by core team members prior to deployment. For significant updates, Omnipair engages security auditors for thorough review.

<Note>
Omnipair does not currently employ a timelock for upgrades. Timelock mechanisms are planned as the protocol moves toward further decentralization and community governance.
</Note>
