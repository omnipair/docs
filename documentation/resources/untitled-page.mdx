## General

<AccordionGroup>
  <Accordion title="What is Omnipair?">
    Omnipair is a decentralized protocol for spot trading, borrowing, and leverage built around isolated, pair-specific money markets. Each market operates independently and supports swaps, collateralized borrowing, leveraged positions, and liquidity provision within a single pool.

    Unlike traditional AMMs or lending protocols, Omnipair does not rely on governance whitelists or external price oracles. Any SPL token pair with liquidity can form its own market, with risk and pricing determined by that market’s dynamics.
  </Accordion>
  <Accordion title="How is Omnipair different from a traditional AMM or lending protocol?">
    Traditional AMMs focus only on swaps, while lending protocols typically support a limited set of approved assets with shared risk across markets. Omnipair unifies swaps, lending, and leverage into isolated markets where each token pair defines its own risk, pricing, and borrowing capacity.

    This design allows long-tail and emerging assets to form money markets organically, without exposing the rest of the protocol to their risk.
  </Accordion>
  <Accordion title="Do I need a wallet to use Omnipair?">
    Yes. Omnipair is non-custodial, so you’ll need a Solana-compatible wallet (such as Phantom or Solflare) to interact with the protocol.
  </Accordion>
  <Accordion title="What blockchains does Omnipair support?">
    Omnipair is currently deployed on Solana and  leverages Solana’s high throughput and low transaction costs.
  </Accordion>
  <Accordion title="Can anyone create a market on Omnipair?">
    Yes. Any user can permissionlessly create a market for any SPL token pair by supplying initial liquidity and selecting risk parameters.c
  </Accordion>
  <Accordion title="How do I interact with Omnipair?">
    Users can interact with Omnipair through a user-friendly web interface or directly with its on-chain programs on Solana. This openness also enables developers to build third-party applications, integrations, and services that compose on top of Omnipair’s permissionless markets.

    To interact with Omnipair through the interface, simply connect a Solana-compatible wallet and select a market. From there, you can swap assets, provide liquidity, borrow against collateral, or create leveraged positions within that market. Assets supplied to a pool earn fees and interest based on market activity, and supplied collateral can be used to access borrowing and leverage within the same isolated pool.
  </Accordion>
  <Accordion title="What is the cost of interacting with Omnipair?">
    Interacting with the Omnipair Protocol requires transactions on Solana. These transactions may incur gas fees, which are non-refundable network transaction fees determined by the network status and transaction complexity.
  </Accordion>
</AccordionGroup>
